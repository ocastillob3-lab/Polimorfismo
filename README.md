# 🧠 Tarea de Polimorfismo en Java

## 📌 Descripción

En este proyecto trabajé el tema de **polimorfismo en Java**, aplicando herencia y sobrescritura de métodos.

La idea fue demostrar que una variable del tipo de la clase padre puede apuntar a un objeto de la clase hija, y que el método que se ejecuta depende del objeto real.

---

## 🎯 Objetivo

Comprender cómo funciona el polimorfismo utilizando:

* Herencia (`extends`)
* Sobrescritura de métodos (`@Override`)
* Uso de referencias del tipo base

---

## ⚙️ Tecnologías utilizadas

* Java
* Eclipse IDE
* GitHub

---

## 🧬 ¿Qué hice?

* Creé clases padre e hijas
* Apliqué herencia con `extends`
* Sobrescribí métodos usando `@Override`
* En el `main` usé polimorfismo

---

## 📂 Estructura del proyecto

El proyecto contiene:

* 10 clases padre
* 10 clases hijas
* 1 clase principal (`Main`)

Todas las clases están separadas en diferentes archivos dentro del mismo paquete.

---

## 🚀 Ejecución

1. Abrir el proyecto en Eclipse
2. Ejecutar la clase `Main`
3. Ver los resultados en consola

---

## 📝 Conclusión

Con esta práctica entendí que:

* El polimorfismo permite reutilizar código
* El método que se ejecuta depende del objeto real
* Se debe usar herencia y sobrescritura correctamente
